/ 02 · 界面

四个面板，
一条河。

01
创作 / Script
从一段散文式的草稿出发，每一句都可以成为镜头。Nomi 会读懂上下文，给你提示词、节奏、灵感。
02
生成 / Canvas
一个无限画布。角色定妆、场景、镜头分层成可编辑的卡片，连成你想要的故事，跨节点复用参考——下方就是吸附剪辑的镜头时间轴。
03
3D 导演台 / Director
文字说不清的构图，直接「搭」出来：摆人偶姿势、架机位、设画幅，截机位画面当生成参考，锁住跨镜头的空间关系。再用一句话让 AI 控运镜。
04
预览 / Timeline
所有节点都已经在时间轴上。拖、剪、叠、调速——传统剪辑该有的事情，都在这里发生。
/ 03 · 为什么这样设计

一条画布，
而不是
七个软件。

剧本、生成、剪辑，本来就该是一件事。把它们拆成三个软件，是工具的局限，不是创作的局限。Nomi 把它们压回同一张画布——剧本里的提示词直接长出图像和视频节点，节点的片段再直接落进剪辑轨。

"数据丝滑流动，而不是在工具之间复制粘贴。"

工程属于你，存在本地。剧本、生成的图片视频、导入的素材都落进你选定的本地文件夹，换台电脑拷走就走。AI 生成走你自己的供应商账户：用谁、花多少，你说了算。

Agent 是助理，不是替身。它会读上下文、补镜头、提建议，但 Nomi 不会替你做决定——所有的"选哪个"，留给你按下去。

/ 04 · 这里没有什么魔法

剧本 写出来 → 提示词 自动抽出 →
生成 图像 / 视频 节点 →
节点 拖入 时间轴 → 一条 成片。

一条画布，
从头到尾。
