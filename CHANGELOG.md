# 更新日志

本项目的所有重要变更记录于此。版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## v0.12.1 — 修复自动更新（2026-06-22）

修复旧版本「检查更新」报错、无法升级的问题。

### 🔧 自动更新
- **macOS 打包补回 `zip` 产物**：此前 mac 只打 `dmg`，`latest-mac.yml` 缺 zip，旧版本检查更新时报 `ZIP file not provided`、彻底卡死。补回 zip 后报错消除，Windows 端恢复就地自动更新。
- **macOS 手动更新兜底**：应用尚未做代码签名，而 macOS 的 Squirrel 强制校验签名才能就地装。现在 mac 检测到新版后给「前往下载」直达 GitHub 最新 release 手动重装，并明确标注原因，不再静默失败。

## v0.12.0 — 导演工作台（2026-06-22）

自 v0.10.13 起积累的一次大版本：Nomi 从「写故事 → 生成 → 剪片」的流水线，长成了一个能**把你脑子里的画面精确传达给 AI** 的导演工作台——3D 摆构图、运镜控节奏、一键被编程助手驱动。

### 🎬 3D 导演台 + AI 运镜（全新支柱）
- **3D 导演台**：在 3D 场景里摆角色姿势、架机位、设画幅，截机位画面当生成参考——文字说不清的构图直接「搭」出来，锁住跨镜头的空间关系。内置 12 个校准过的预设动作，姿势后自动落地（不悬空）。
- **AI 站位参考**：让 AI 按你的描述自动搭好站位 / 动作 / 朝向 / 机位，离屏出图、连成参考，锁站位锁身份。
- **AI 运镜**：对话里描述运镜 → 沿相机轨迹渲一段运镜小片 → 喂进 Seedance 2.0 全能参考（video_ref），把运镜复刻到真实视频上。轨迹编辑器已挂进 3D 编辑器，人也能手搓。

### 🔗 一键接入编程助手（能力核）
- Nomi 的创作能力抽成无头能力核（CLI / MCP）：在「模型接入」面板一键把 Nomi 接进 **Claude Code / Codex / Cursor**，零配置写好接入，用对话指挥建项目、加节点、连参考、触发生成。
- MCP 驱动 Nomi 所见即所得；Nomi 关着时也能在 Claude 里用 elicitation 确认付费生成，不死锁。
- 握手回显客户端协议版本，兼容所有 MCP 客户端。

### 🤖 模型接入更省心
- 模型接入页简化：分区列表 + 顶部能力概览，一眼看清「已通 / 半通 / 断」。
- 免配置上传做成多床冗余链（litterbox → tmpfiles），运镜 mp4 喂入彻底不用 key。

### 🎞️ 时间轴重做
- clip 用真实帧缩略图，一眼辨识。
- 重做（⇧⌘Z）、trim 帧气泡、一键拼片、选中 clip 就地「重生成」。
- 播放器补音量 / 静音 / 全屏 / 逐帧。

### ⚡ 性能 + 稳定性
- 冷启动 + 流式 + 画布全链路卡顿三处根治；窗口不再被代理探测 / 能力核阻塞。
- 损坏项目优雅降级——缺字段不再让项目硬锁死打不开。
- 提示词库缓存落盘。

### 🛡️ 设计系统 + 字体
- 用户可见面全面对齐设计系统 v2；重接被切断的设计 token + 新增悬空 token 门岗。
