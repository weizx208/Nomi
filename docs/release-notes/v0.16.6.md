# Nomi v0.16.6 — 应用内浏览器 + 全局素材盒（2026-07-12）

> 主角是社区贡献者 @1251912798 的 18k 行大 PR（#36）：Nomi 里长出一个真浏览器。
> 我们在合入时做了安全/隐私/一致性收口，并把原「网页捕捞」独立窗收敛进同一引擎。

## 用户可感变化

1. **顶栏「浏览器」**：多标签 + 书签 + 素材网站快捷入口。找参考全程不出 Nomi。
2. **素材盒（顶栏）**：素材收件箱。开「资源捕捞」后**悬停图片按 Ctrl+C** 或**拖拽**即入当前项目素材库，徽章实时计数；多选可一把拖上画布。
3. **网页偷师**：划词存提示词；选区截图 → AI 提取提示词/复刻风格。
4. **「网页捕捞」按钮还在素材库**，点开的就是新浏览器（原独立捕捞窗退役；旧窗登录过的站需重登一次）。
5. **顶栏/左栏统一**：操作全部图标+文字；左栏功能条带微字标签。
6. **隐私**：捕捞素材只存本地，浏览的网页地址绝不发给生成服务商。
