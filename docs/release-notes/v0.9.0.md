# v0.9.0 — 时间轴交互重做 + 音频导出

发布日期：2026-06-04

## 一句话

**剪辑手感重做，导出开始有声音。**

时间轴从手搓拖拽升级为「靠近吸附、拖远脱离」的精细剪辑，播放头能拖、片段能多选成组；导出主路径切到 FFmpeg filtergraph，直读源文件出片，开始带上视频自带的音轨，且预览=成片（letterbox 一致）。

## 新增 / 改进

### 时间轴交互（P1）

- **拖拽自动吸附**：拖片段靠近相邻片段边 / 播放头 / 起点自动吸附（暖橙辅助线 + 轻微咔哒），**拖远即脱离，无需快捷键**；整秒吸附默认关，避免"搓衣板感"。
- **可拖播放头**：直接拖蓝色把手 scrub，点标尺跳转并可续拖；按帧对齐。
- **多选**：Shift/⌘ 点选多个片段，**成组拖动**（整体夹紧不与其它片段重叠），批量删除；点轨道空白清空选择。
- **撞了滑入最近空位**，不再弹回原位；trim 两端手柄也吸附。
- 修**播放漂移**：播放推进从固定定时器改为按真实墙钟时间推进（`requestAnimationFrame`），后台标签页不再跑飞、支持播放中 scrub。
- trim 手柄改为左右对称握把（修掉之前左窄右宽）。

### 导出（P3）

- **导出开始带音频**：导出主路径切到 FFmpeg filtergraph，直读本地源文件渲染；视频自带的音轨随片导出，多段自动 amix 混音（`normalize=0` 防衰减）。
  - 实验性：解析/编码失败会**自动回退**到原 WebM 转码（静音），导出不中断、无回归。
- **预览=成片（WYSIWYG）**：导出几何统一为 letterbox（留黑边、整帧不裁），与预览一致。

### 文档

- README 修正：导出格式标注为 MP4（此前误写 WebM）；时间轴描述同步新能力。

## 已知限制 / 后续

- 同一素材被多个片段复用时的音/视频流重复引用为既有限制，后续用 split/asplit 解决。
- 「权威视频时钟」级别的严格音画同步属后续。
