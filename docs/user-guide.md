# Nomi 使用指南

## 基本流程

```
创作区 → 生成区（画布）→ 时间轴 → 导出
```

---

## 创作区

写剧本、章节、镜头描述。

- 左侧面板选择项目和章节
- 直接在编辑器里写内容
- 内容自动保存到本地（`文档/Nomi Projects`）

---

## 生成区（画布）

把文字变成图片和视频。

**添加节点**
点击工具栏 `+` → 选择节点类型（图片 / 视频 / 文本 / 角色 / 场景等）

**生成图片**
1. 点击节点选中
2. 在下方输入框写提示词
3. 点击「生成 →」

**生成视频**
视频节点需要先连接一个图片节点作为首帧：
1. 点击图片节点右侧的连接点
2. 拖线到视频节点左侧
3. 填写提示词 → 点击「生成 →」

**连接节点**
点击节点边缘的圆点，拖到另一个节点。

**快捷键**
| 操作 | 快捷键 |
|------|--------|
| 复制节点 | Cmd/Ctrl + D |
| 删除节点 | Delete |
| 适应视图 | 工具栏 ⌖ 按钮 |
| 平移画布 | 拖拽空白区域 |

---

## 时间轴

把生成好的素材排成视频。

**添加素材**
从生成区把节点拖到时间轴轨道上。

**编辑片段**
| 操作 | 方式 |
|------|------|
| 移动片段 | 拖拽 |
| 裁剪 | 鼠标悬停在片段边缘，拖动 handle |
| 分割 | 选中片段，按 `S` |
| 复制 | 选中片段，按 Cmd/Ctrl + D |
| 删除 | 选中片段，按 Delete |
| 微移 | Shift + `<` / `>` |

**缩放时间轴**
Ctrl/Cmd + 滚轮，或点击工具栏 +/- 按钮。

**播放预览**
点击播放按钮，或按空格键。

---

## 导出

点击预览区右上角「导出 WebM」。

- 格式：WebM（VP9）
- 分辨率：1280px 宽
- 注意：**不包含音频**

导出完成后自动保存到本地。

---

## 终端 Agent（高级）

在终端里用自然语言控制整个工作台。

**配置**

在 `apps/agents/` 目录下创建 `agents.config.json`：

```json
{
  "apiBaseUrl": "https://api.deepseek.com/v1",
  "apiKey": "你的 API Key",
  "model": "deepseek-chat"
}
```

**启动**

```bash
cd apps/agents
pnpm install
pnpm dev
```

**示例指令**

```
把这段剧本拆成 6 个镜头，每个镜头生成图片，按顺序放进时间轴。
```

```
读取当前画布，把所有图片节点连接到对应的视频节点，然后全部生成。
```

```
在时间轴第 3 个片段后面插入一个 3 秒的黑场。
```

Agent 执行时，画布会实时展示变化。

**可用工具**

| 工具 | 说明 |
|------|------|
| `canvas_read` | 读取画布节点和连线 |
| `canvas_create_nodes` | 创建节点 |
| `canvas_run_node` | 触发生成 |
| `timeline_read` | 读取时间轴 |
| `timeline_add_clip` | 添加片段 |
| `creation_read` | 读取文稿 |
